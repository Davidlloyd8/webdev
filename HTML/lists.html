LIST

There are lots of occasions when we need to use lists. HTML provides us with three different types:

  • Ordered Lists: This are lists where each item in the list is numbered. For example, the list might be a set of steps for a recipe that must be performed in order, or a legal contract where each point needs to be identified by a section number.
  • Unordered Lists: This are lists that begin with a bullet point (rather than characters that indicate order).
  • Definition Lists: This are made up of a set of terms along with the definitions for each of those terms.

LIST ITEM

Each item in the list is placed between an opening <li>tag and a closing </li> tag. (The li stands for list item.)

Ordered List

The ordered list is created with the <ol> element.

NB: Sometimes you may see a type attribute used with the <ol> element to specify the type of numbering (numbers, letters, roman numerals and so on). It is better to use the CSS list-style-type property.

Unordered List

The unordered list is created with the <ul> element.

NB: Sometimes you may see a type attribute used with the <ul> element to specify the type of bullet point (circles, squares, diamonds and so on). It is better to use the CSS list-style-type property

Definition List

The definition list is created with the <dl> element and usually consists of a series of terms and their definitions.

Definition Term: This is used to contain the term being defined (<dt>).
Definition Data: This is used to contain the definition.

HORIZONTAL LINES
Horizontal lines are used to visually break up sections of a document. The horizontal line tag create a line from the current position in the document to the right margin and breaks the line accordingly.
PARAGRAPH
The paragraph tag offers a way to structure your text into different paragraphs.
