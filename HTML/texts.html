TEXT

When creating a web page, you add tags (known as markup) to the contents of the page. These tags provide extra meaning and allow browsers to show users the appropiate structure for the page.
There are two ways to add markup to your text that appears on your page:

  • Structural markup: the elements that you can use to describe both headings and paragraphs
  • Semantic markup: This is a way of writing and structuring your HTML (Hypertext Markup Language) so that it reinforces the semantics, or meaning, of the content rather than its appearance.

HEADINGS

HTML has six levels of headings: <h1></h1> to <h6></h6>
<h1> : is used for main headings
<h2> : is used for subheadings and so on
NB: Browsers display the contents of headings at different sizes. The contents of an <h1> element is the largest, and the contents of an <h6> element is the smallest. The exact size at which each browser shows the headings can vary slightly
