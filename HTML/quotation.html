QUOTATION

There are two elements commonly used for marking up quotations:

  • Blockquote: The <blockquote> element is used for longer quotes that take up an entire paragraph.
    NB: Browsers tend to indent the contents of the <blockquote> element, however you should not use this element just to indent a piece of text â€” rather you should achieve this effect using CSS.
  • Quote: The <q> element is used for shorter quotes that sit within a paragraph.
    NB: Browsers are to put quotes around the <q> element
