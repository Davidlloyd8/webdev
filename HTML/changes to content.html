CHANGES TO CONTENT

INSERTED

The <ins> element can be used to show content that has been inserted into a document.
NB: The content of a <ins> element is usually underlined,

DELETED

The <del> element can show text that has been deleted from it.
NB: The content of a <del> element usually has a line through it.

SELECTED

The <s> element indicates something that is no longer accurate or relevant (but that should not be deleted).
NB: Visually the content of an <s>element will usually be displayed with a line through the center.
